Resistor Calculator

The electronic color code is used to indicate the values or ratings of electronic components, very commonly for resistors. There are lots of different implementations online already, but we believe you can do better. We've been hard at work on the back-end code and now your goal is to knock our socks off with a user interface that implements that!

Provided are two implementations of the calculator: a plain Javascript version and a Node.js module. They are essentially identical and you can use either one. Pick whichever one is more convenient for you.

Use your imagination! Get creative! All we ask is that your submission be self-contained and easy to view and use without requiring us to install a lot of tools or frameworks. Place your submission in a public Github repository and send us the clone URL. If you have questions, don't hesitate to ask. Good luck!
